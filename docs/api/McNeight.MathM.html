Class MathM

Provides constants and static methods for trigonometric, logarithmic, and other common mathematical functions.

public static class MathM
Remarks

The static fields and methods of the MathM class correspond to those of the System.MathF classes, except that their parameters are of type System.Decimal rather than System.Single, and they return System.Decimal rather than System.Single values.

Type Description

An angle, θ, measured in radians, such that -π ≤ θ ≤ π, and tan(θ) = y / x, where (x, y) is a point in the Cartesian plane. Observe the following: For (x, y) in quadrant 1, 0 < θ < π/2.
